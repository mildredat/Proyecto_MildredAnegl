# Proyecto_MildredAnegl
BIO-TECH es un sistema de gestión biomédica desarrollado con Node.js, Express y MySQL. Administra usuarios, órdenes de estudio y resultados clínicos bajo un entorno seguro con roles (Admin, Médico, Paciente). Incluye carga/descarga masiva de datos vía Excel y validación de registros mediante códigos únicos.
